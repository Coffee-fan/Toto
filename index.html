Totoby JeremyOT

A simple Tornado based framework designed to accelerate web service development

Dependencies

This project depends on Tornado (http://www.tornadoweb.org/) for base functionality. Optionally, pymongo is required when using built in MongoDB or BSON functionality. MySQL-python is required for use with MySQL.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/JeremyOT/Toto

Install

The simplest way to install Toto is with pip. Simply run 

pip install toto
to install the latest stable version of the Toto module on your machine. Alternatively, you can run 
 pip install -e git+git://github.com/JeremyOT/Toto.gitEegg=Toto
to install directly from the latest source (may not be stable).
